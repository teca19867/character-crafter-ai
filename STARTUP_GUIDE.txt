Character Crafter AI - Startup Guide
=====================================

Quick Start:
1. Double-click start.bat (Recommended) - Full check and startup
2. Double-click quick-start.bat - Simple quick start

Manual Startup:
1. Open Command Prompt or PowerShell
2. Navigate to project directory
3. Run: npm run dev

Notes:
- First run will automatically install dependencies
- Requires Node.js environment
- Application will open at http://localhost:3000
- Press Ctrl+C to stop the server

API Configuration:
- To use AI features, follow these steps:
  1. Copy .env.example file to .env
  2. Edit .env file, replace your_google_gemini_api_key_here with your actual API key
  3. Get API key: https://aistudio.google.com/app/apikey

Security Reminder:
- Please ensure not to commit .env files containing real API keys to version control
- For third-party APIs, configure them in the app's settings panel
